Node.js 버전 변경하는 방법 (MAC)

이 포스트는 npm package 중 n 을 이용합니다.

Mac 에서 동작하는 방법이며, Windows Shell 또는 Git for Windows 에서는 동작하지 않습니다.

Node 버전 변경이 필요한 이유​

Node.js 서버를 통해 이것저것 하다보면 버전의 영향을 많이 받을 수도 있습니다.

예를 들어, serverless, node-sass, node-gyp 등의 오류를 피해야 하는 문제가 생깁니다.

오늘은 n 을 통하여 설치된 Node.js 버전을 변경하는 방법을 알아보도록 합니다.

필독! 이미 설치된 경우,​

Homebrew를 통해 /opt/homebrew/Cellar/node/{version}/ 에 Node.js가 설치된 경우, n 과는 호환이 되지 않으므로 기존 Node.js를 지워줘야 합니다.

Node.js 제거 방법​

Node.js 홈페이지를 통해서 다운로드 한 경우가 아닌 Homebrew 나 기타 방법으로 설치를 받은 경우에 해당됩니다.

Node.js 깔끔하게 제거하는 방법 (MAC)​

  1. Node.js 홈페이지에서 받았을 때,
    • sudo rm -rf /usr/local/lib/node
    • sudo rm -rf /usr/local/lib/node_modules
    • sudo rm /usr/local/lib/dtrace/node.d
    • sudo rm /usr/local/share/man/man1/node.1
    • sudo rm -rf /usr/local/

Node.js 설치 방법​

  1. Node.js 웹사이트 접속
    • Download URL : https://nodejs.org/ko/download/
    • LTS 버전 : Long Term Support 의 약자 최신 버전보다는 안정화 버전으로 받는 것을 추천

Node.js 버전 변경 방법​

  1. npm 을 통해 n 을 전역에 설치해줍니다.
    • sudo npm install -g n
  2. 현재 설치된 Node.js 버전 확인
    • sudo n ls
  3. 추가로 필요한 Node.js 버전 설치
    • sudo n install <버전>
  4. 활성화시킬 Node.js 버전 변경
    • sudo n
    • 키보드의 화살표를 움직여 활성화시킬 Node.js 버전을 선택해줍니다.

주요 명령어 정리​

  • sudo n : 활성화시킬 Node.js 버전 변경
  • sudo n ls : 현재 설치된 Node.js 버전 확인
  • sudo n install <버전> : 특정 버전의 Node.js 설치
  • sudo n rm <버전> : 특정 버전의 Node.js 제거
  • n 명령어1 앞에 sudo 를 붙이는 이유는 Node.js 를 관리하는 /usr/local/ 경로에 접근하고 파일을 설치, 수정, 삭제하기 위해서입니다.
  1. 기타 궁금한 명령어는 n --help 를 통해 확인할 수 있습니다.↩
