블로그 새로 시작하기

2023년 8월 29일 · 약 2분

재테크, 부동산, 건강 등 여러 블로그를 운영하고 있지만, 가장 운영에 고민이 되는 블로그는 항상 개발 블로그인것 같습니다.

블로그 리뉴얼​

다른 주제의 블로그의 경우, 정보성 글만 올리다보니 정확도만 신경 쓰면 되는 편입니다. 반면에 개발 블로그는 항상 꾸준히 내가 학습하고 경험해 본 지식들을 적어야하는 부담감이 있습니다. 정확도를 신경 쓰는만큼 꾸준히 개발하고 그 일들을 기록하는 일들이 그만큼 쉽지 않다는 것이지요.

블로그... 였던 것​

Github Blog, Tistory, Velog 이곳저곳 떠돌아다니면서 포스팅을 했으나, 어디 하나 자리잡고 꾸준히 적지 못하고 만들고 지우고를 반복했던 흔적이 많습니다.

Github Pages 재개발​

일전에 Jekyll, Gatsby 등등 여러 시도를 해보았으나, 디자인만 고르다가 결국 포기하기를 수도 없이 반복했던 것 같습니다 .

Docusaurus 라는 Meta의 라이브러리가 생긴 것을 확인하고 Dyte Docs의 템플릿을 사용하기로 했습니다.

Docusaurus는 개발 문서와 블로그를 같이 잘 관리할 수 있는 라이브러리인 것 같습니다. 이왕 마음먹고 다시 시작한 김에 프로필, 포트폴리오, 이력서, 블로그를 한꺼번에 통합해야겠다는 생각으로 다시 만들고 있습니다.

블로그의 핵심은 역시나 글인지라 꾸준함이 또 필요하겠지요 ㅠㅠ

마무리​

아직 글 솜씨가 좋지는 못해서 말투나 문장을 적는 습관이 좋진 않은 것 같지만, 이왕 개인 문서를 다 모아 놓는다고 생각하고 꾸준히 해봐야겠습니다.
